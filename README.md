Описание файлов

X_bp.xlsx - Файл с первой частью исходных данных

X_nup.xlsx - Файл со второй частью исходных данных

X_united_full.xlsx - Файл после предварительной обработки и объединения исходных наборов данных


VKR_UMN_Preprocc#1.ipynb - Предобработка исходных данных

VKR_UMN_EDA#1.ipynb - Разведочный анализ предобработанных данных

VKR_UMN_ML_Modul#1.ipynb - Обучение модели для прогноза модуля упругости при растяжении КМ

VKR_UMN_ML_Streng#1.ipynb - Обучение модели для прогноза прочности при растяжении КМ

Matrix_NN.ipynb - Обучение нейронной сети для рекомендации соотношения матрица-наполнитель в составе разрабатываемых КМ


ВКР Усачёв.docx - Пояснительная записка к проекту

ВКР_Усачёв М.Н.pptx - презентация к проекту
